How Grad-CAM Works

Class Activation Mapping via gradient flow — Selvaraju et al. 2017.

Forward Pass
Forward hook on the last Conv2d layer captures feature maps Ak ∈ ℝh×w.
Backward Pass
Backward hook captures gradients ∂yc/∂Ak for the target class score yc.
Importance Weights
Global-average-pool gradients: αk = (1/Z) Σij ∂yc/∂Akij.
Weighted Sum + ReLU
L = ReLU(Σk αk · Ak), upsample to 224×224.
Overlay
Jet colormap applied → blended α=0.45 over grayscale X-ray. Red = highest activation.
